# Analytics Service

Responsible for processing raw academic data into meaningful KPIs and chart-ready formats for the administrative dashboard.

## Features
- Calculates occupancy percentages and student distribution.
- Generates data for Pie Charts (Status distribution) and Bar Charts (Critical Careers).
- Provides administrative KPIs: Total Students, Saturated Courses, and Average Occupancy.

## Tech Stack
- **Framework:** NestJS
- **ORM:** Prisma (PostgreSQL)

## Dependencies
- **PostgreSQL:** Reads real-time data from the shared academic database.
